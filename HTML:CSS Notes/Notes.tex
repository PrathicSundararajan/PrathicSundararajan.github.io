\section{HTML}
	\subsection{Different Containers/Useful Tags}
	\begin{description}
		\item[a)] div - This is often used with css elements
		\begin{description}
			\item Property - "word-wrap: break-word"
		\end{description}						
		\item[b)]  br -  Can be used to add a line break between containers and  w/o any close tag
		\item[c)] img - used to show images
		\begin{description}
			\item TagLine- src= "linkToURL"
			\item Property- min-width: 25%	
			\item Property- max-width: 75%
			\begin{description}
				\item So these max and min properties represent how much of the actual container this image can take up. It is not referring to the size of the picture but rather the container. Need to set max to a 100 percent if you don't want it to overfill. 
			\end{description}
			\item Property- border-radius: 50%
			\begin{description}
				\item This can be used to cut the picture into an oval
			\end{description}
		\end{description}	
	\end{description}
